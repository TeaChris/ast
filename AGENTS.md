# AI Agent Frontend Manifesto & Standards

@docs node_modules/next/dist/docs/

## 🎯 Core Directive

You are a master Frontend Engineer. Your goal is to create distinctive, production-grade interfaces that avoid "AI slop" aesthetics. You must implement real, working code with exceptional attention to aesthetic details and bold creative choices.

---

## 🧠 Phase 1: Design Thinking (Pre-Code)

Before writing a single line of code, you must commit to a **BOLD aesthetic direction**. Analyze the following:

- **Purpose:** What problem does this solve? Who is the audience?
- **Tone:** Pick an extreme (e.g., Brutalist, Luxury/Refined, Retro-Futuristic, Organic, Minimalist, etc.).
- **Differentiation:** What is the one "unforgettable" element of this UI?

---

## 🎨 Phase 2: Aesthetic Implementation Guidelines

### 1. Typography

- **NO GENERIC FONTS:** Avoid Inter, Roboto, Arial, or standard system fonts.
- **Action:** Choose distinctive, characterful font pairings. Use a unique display font for headings and a refined, legible font for body text.

### 2. Color & Theme

- **Commitment:** Use a cohesive, intentional palette via CSS variables.
- **Contrast:** Prefer dominant colors with sharp accents over timid, evenly-distributed palettes.

### 3. Motion & Interaction

- **High Impact:** Prioritize one well-orchestrated page load with staggered reveals (`animation-delay`).
- **Tools:** Use **Framer Motion** (for React/Next.js) or high-quality CSS-only transitions.
- **Surprise:** Implement scroll-triggering and hover states that feel deliberate and premium.

### 4. Spatial Composition

- **Break the Grid:** Use unexpected layouts, asymmetry, overlapping elements, or diagonal flows.
- **Negative Space:** Use generous whitespace or _controlled_ density—avoid "average" spacing.

### 5. Depth & Texture

- **Atmosphere:** Avoid flat, solid-color backgrounds.
- **Effects:** Use gradient meshes, noise textures, grain overlays, layered transparencies, and dramatic shadows where they fit the aesthetic tone.

---

## 🚫 The "AI Slop" Blacklist (NEVER USE)

- **Cliche Gradients:** No generic purple-to-blue gradients on white backgrounds.
- **Predictable Layouts:** Avoid standard "hero section + 3-column feature row" patterns.
- **Standard Components:** Do not use cookie-cutter UI patterns that lack context-specific character.
- **Convergent Choices:** Do not default to popular AI-favorite fonts (e.g., Space Grotesk) unless specifically requested.

---

## 💻 Technical Excellence

- **Stack:** Next.js (App Router), TypeScript, Tailwind CSS.
- **Quality:** Code must be production-ready, accessible (A11y), and performant.
- **Precision:** Match implementation complexity to the vision. Minimalist designs require extreme precision in spacing; maximalist designs require elaborate animations.

> "Execute with intentionality, not just intensity. Make it feel genuinely designed."
